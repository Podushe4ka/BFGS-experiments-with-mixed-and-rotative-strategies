# BFGS-experiments-with-mixed-and-rotative-strategies

Барышев Даниил Владимирович, 3 курс, ПАДИИ, ВШЭ СПБ
----
В данной работе рассматриваются методы обновления приближенной матрицы обратного гессиана в семестве методов BFGS из работ ученых [Пекинского и Сианского университетов](https://jmlr.org/papers/volume23/21-1282/21-1282.pdf) и ставятся следующие задачи:

1. Реализация алгоритмов, сравнение результатов работы на выбранных датасетах с определенными функционалами
2. Формализация критерия выбора направления спуска, выработка критерия и оценка его эффективности
3. Теоретический анализ оценки сходимости
4. Эксперименты, сравнения, анализ

Выбранные датасеты для экспериментов: [w8a](https://www.csie.ntu.edu.tw/~cjlin/libsvmtools/datasets/binary/w8a), [E2006](https://www.csie.ntu.edu.tw/~cjlin/libsvmtools/datasets/regression/E2006.train.bz2)
