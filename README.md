# BFGS-experiments-with-mixed-and-rotative-strategies

Барышев Даниил Владимирович, 3 курс, ПАДИИ, ВШЭ СПБ
----
В данной работе рассматриваются подходы обновления приближенной матрицы в семестве методов BFGS из работ ученых [Пекинского и Сианского университетов](https://jmlr.org/papers/volume23/21-1282/21-1282.pdf) и ставятся следующие вопросы:

1. Можно ли улучшить количество итераций в меньшую сторону с помощью комбинирования двух методов (рандомная ротация, выработка критерия)
2. Если да, то как можно сформулировать критерий, формализация мотивировки в контексте определенных методов.
3. Сравнение с классическим BFGS и L-BFGS
4. Сохранилась ли сверхлинейная локальная сходимость для полученного метода

Выбранные датасеты для экспериментов: [w8a](https://www.csie.ntu.edu.tw/~cjlin/libsvmtools/datasets/binary/w8a), [E2006](https://www.csie.ntu.edu.tw/~cjlin/libsvmtools/datasets/regression/E2006.train.bz2)
